## Get All Treatments

Retrieve a list of all treatment records.

### Endpoint

```
GET /treatments
```

### Response

<ResponseField name="treatments" type="array">
  Array of treatment objects
  
  <ResponseField name="id" type="integer">
    Unique identifier for the treatment
  </ResponseField>
  
  <ResponseField name="childId" type="integer">
    ID of the child receiving the treatment
  </ResponseField>
  
  <ResponseField name="medicationId" type="integer">
    ID of the medication used in the treatment
  </ResponseField>
  
  <ResponseField name="startDate" type="date">
    Date when the treatment begins
  </ResponseField>
  
  <ResponseField name="endDate" type="date">
    Date when the treatment ends
  </ResponseField>
  
  <ResponseField name="specificDose" type="integer">
    Specific dose amount for this treatment
  </ResponseField>
  
  <ResponseField name="instructions" type="string">
    Additional instructions for administering the treatment
  </ResponseField>
</ResponseField>

### Example Request

```bash
curl -X GET http://localhost:8080/treatments
```

### Example Response

```json
[
  {
    "id": 1,
    "childId": 5,
    "medicationId": 1,
    "startDate": "2026-03-01T00:00:00.000+00:00",
    "endDate": "2026-03-15T00:00:00.000+00:00",
    "specificDose": 250,
    "instructions": "Take with food. Do not exceed 3 doses per day."
  },
  {
    "id": 2,
    "childId": 7,
    "medicationId": 2,
    "startDate": "2026-02-25T00:00:00.000+00:00",
    "endDate": "2026-03-10T00:00:00.000+00:00",
    "specificDose": 500,
    "instructions": "Complete full course even if symptoms improve."
  }
]
```

### Status Codes

| Code | Description |
|------|-------------|
| `200` | Successfully retrieved treatments |

## Get Treatment by ID

Retrieve details of a specific treatment.

### Endpoint

```
GET /treatments/{id}
```

### Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the treatment
</ParamField>

### Response

<ResponseField name="id" type="integer">
  Unique identifier for the treatment
</ResponseField>

<ResponseField name="childId" type="integer">
  ID of the child receiving the treatment
</ResponseField>

<ResponseField name="medicationId" type="integer">
  ID of the medication used in the treatment
</ResponseField>

<ResponseField name="startDate" type="date">
  Date when the treatment begins
</ResponseField>

<ResponseField name="endDate" type="date">
  Date when the treatment ends
</ResponseField>

<ResponseField name="specificDose" type="integer">
  Specific dose amount for this treatment
</ResponseField>

<ResponseField name="instructions" type="string">
  Additional instructions for administering the treatment
</ResponseField>

### Example Request

```bash
curl -X GET http://localhost:8080/treatments/1
```

### Example Response

```json
{
  "id": 1,
  "childId": 5,
  "medicationId": 1,
  "startDate": "2026-03-01T00:00:00.000+00:00",
  "endDate": "2026-03-15T00:00:00.000+00:00",
  "specificDose": 250,
  "instructions": "Take with food. Do not exceed 3 doses per day."
}
```

### Status Codes

| Code | Description |
|------|-------------|
| `200` | Treatment found and returned |
| `404` | Treatment not found |
